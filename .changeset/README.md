# Changesets

Hello and welcome! This folder has been automatically generated by `@changesets/cli`, a build tool that works
with multi-package repos, or single-package repos to help you version and publish your code. You can
find the full documentation for it [in the repository](https://github.com/changesets/changesets)

## Usage

To add a changeset, run `bunx changeset` in the root of the repository. Follow the prompts to describe your change.

When changes are merged to `main`, a "Version Packages" PR will be automatically created (or updated).
Merging that PR will publish the new version to npm.
